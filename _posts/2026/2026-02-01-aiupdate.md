## [META](https://ai.meta.com/research/)

* 🌟 [DINOV3](https://ai.meta.com/research/dinov3/) **scales self-supervised learning** (SSL) for images to produce our strongest universal vision **backbones**, enabling breakthrough performance across diverse domains.
* 🌟 [SAM(segment-anything) 3](https://ai.meta.com/research/sam3/), is state-of-the-art across **all text and visual segmentation tasks in both images and videos**. The model additionally maintains all the performance and functionality of SAM 2.
* [SAM 3D](https://ai.meta.com/research/sam3d/) SAM 3D contains two state-of-the-art models that enable **3D reconstruction** of objects and humans from a **single image**.
* [Video Joint Embedding Predictive Architecture 2 (V-JEPA 2)](https://ai.meta.com/research/vjepa/) is the first world model trained on video that achieves state-of-the-art **visual understanding and prediction**, enabling zero-shot robot control in new environments.
  * **Understanding**: A world model should be able to understand observations of the world, including things like recognizing objects, actions, and motions in a video.
  * **Predicting**: A world model should be able to make predictions about how the world will evolve, and how the world will change if the agent takes an action.
  * **Planning**: Building on the ability to make predictions, a world model should be useful for planning sequences of actions that achieve a given goal.

* SIMA 2: An agent that plays, reasons, and learns with you in virtual 3D worlds.
* Genie 3: **world simulation**. A general purpose world model that can generate an unprecedented diversity of interactive environments.

## [Apple](https://github.com/orgs/apple/repositories?q=ml-+sort%3Astars)

* ml-stable-diffusion — ready-to-use Stable Diffusion model integration for Apple devices.
* coremltools — central to converting models (including vision networks) to Core ML for iOS/macOS deployment.
* 🌟 [Sharp Monocular View Synthesis in Less Than a Second](https://github.com/apple/ml-sharp) : Given a **single photograph**, SHARP regresses the parameters of a **3D Gaussian** representation of the depicted scene.
* 🌟 [FastVLM](https://github.com/apple/ml-fastvlm): Efficient Vision Encoding for Vision Language Models. **encode image to LLM**, so we could <u>ask question about image</u>.
* 🌟 [Depth Pro](https://github.com/apple/ml-depth-pro): Sharp **Monocular Metric Depth** in Less Than a Second

## [Niantic Labs](https://github.com/nianticlabs)

* 🌟 [ACE: Scene Coordinate Reconstruction](https://nianticlabs.github.io/acezero/) a **learning-based structure-from-motion approach** that estimates camera parameters of sets of images by learning a multi-view consistent, implicit scene representation.
* [spz](https://github.com/nianticlabs/spz) File format for 3D Gaussian splats. About 10x smaller than the PLY equivalent with virtually no perceptible loss in visual quality.
* 🌟 [morpheus](https://github.com/nianticlabs/morpheus): Text-Driven 3D Gaussian Splat Shape and Color Stylization
* [mvsanywhere](https://github.com/nianticlabs/mvsanywhere): Zero Shot Multi-View Stereo, could be **used to regulate 3DGS**.
